WULF Release Note

WULF_OT31_1_2_6 (released 21.06.2016) - WULF 1.2

WULF is a Bootstrap derivative Nokia Look&Feel to be used in internal and external web UIs. It follows Nokia OceanTouch 3.0 (and partly OT3.1) definitions and main design guidance has been: "responsive and light-weight".
